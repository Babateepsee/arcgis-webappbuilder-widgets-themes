# スクリーンショット（3D）

SceneView のスクリーンショットを作成するウィジェットです。

![Screenshot3D](images/screenshot3d.png)

## 機能

* SceneView で選択した範囲のスクリーンショットの作成、ダウンロード

## 動作環境

* Web AppBuilder 2.11
